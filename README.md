## Overview
Pixel Archaeologist is a motion-led portfolio experience built for visual storytelling.  
It blends cinematic sections, typography-driven composition, and fluid interaction to present a single, cohesive brand narrative.
The project is designed to feel premium while staying practical to maintain.

Live site: [https://public.y7xified.dev](https://public.y7xified.dev)

## Tech Stack
- Astro 5
- Sass (SCSS)
- JavaScript (ES6+)
- GSAP
- Lenis

## Features
- Cinematic one-page layout with narrative pacing
- GSAP-powered transitions and reveal choreography
- Smooth scroll behavior with Lenis integration
- Responsive sections tuned for desktop and mobile
- Typography-first visual hierarchy and polished spacing
- Static build output, easy to deploy on modern hosting

## Installation
1. Clone the repository.

```bash
git clone https://github.com/Y7XIFIED/pixel-archaeologist.git
```

2. Move into the project directory.

```bash
cd pixel-archaeologist
```

3. Install dependencies.

```bash
npm install
```

## Usage
Run locally:

```bash
npm run dev
```

Build and preview:

```bash
npm run build
npm run preview
```
